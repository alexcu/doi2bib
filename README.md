doi2bib
=======

### Installation
    git clone https://github.com/davidagraf/doi2bib.git
    cd doi2bib
    npm install -g bower gulp
    npm install && bower install

### Run Development
    gulp serve

### Create Distribution
    gulp dist

### Run Distribution
    node app/server.js

### Testing
    gulp test

### Debugging
    npm install -g iron-node
    gulp debug
